# Library Management System (MySQL)

## Project Overview
This project implements a simple Library Management System using MySQL.  
It manages books, members, and borrowing records.

## Features
- Add books and members
- Track borrowed books
- Show available books
- Borrow and return tracking
- Analytics queries

## Database Tables
1. Books
2. Members
3. BorrowRecords
